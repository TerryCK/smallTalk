	「必須」這條路則完全不同。「必須」則是真正的自己、我們所堅信的東西、以及當我們遵從自己真實想法時所做的事情。它是我們的天性和渴望，也是從我們內心最深處的直覺。「必須」就是當我們不再遵照別人的期望而完全按照自己的真實想法去做事情。我們一旦選擇了「必須」這條路，我們便無需從外部尋求靈感，反之，我們會傾聽自己內心深處的呼喚。
